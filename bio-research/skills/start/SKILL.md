# 바이오 연구 시작

> 익숙하지 않은 자리표시자가 보이거나 어떤 도구가 연결되어 있는지 확인해야 한다면 [CONNECTORS.md](../../CONNECTORS.md)를 참고하세요.

생물학 연구자가 바이오 연구 플러그인에 빠르게 익숙해질 수 있도록 아래 단계를 순서대로 진행하세요.

## 1단계: 환영하기

다음 환영 메시지를 표시합니다.

```
Bio-Research Plugin

생명과학을 위한 AI 연구 비서입니다. 이 플러그인은
문헌 검색, 데이터 분석 파이프라인,
과학 전략을 한곳에 모아 제공합니다.
```

## 2단계: MCP 서버 연결 상태 확인

어떤 MCP 서버가 연결되어 있는지, 그리고 사용할 수 있는 도구가 무엇인지 확인합니다. 결과는 범주별로 묶어 보여주세요.

**문헌 및 데이터 소스:**
- ~~문헌 데이터베이스 - 생의학 문헌 검색
- ~~문헌 데이터베이스 - 사전 인쇄물 접근(생물학 및 의학)
- ~~저널 접근 - 학술 출판물
- ~~데이터 저장소 - 협업 연구 데이터(Sage Bionetworks)

**신약 개발 및 임상:**
- ~~화학 데이터베이스 - 생리활성 화합물 데이터베이스
- ~~약물 표적 데이터베이스 - 약물 표적 발견 플랫폼
- ~~임상시험 데이터베이스 - 임상시험 등록소
- ~~AI 연구 플랫폼 - 생물학용 AI(조직병리, 신약 개발)

**시각화:**
- ~~과학 일러스트레이션 - 과학 도표와 다이어그램 생성

연결된 서버와 아직 설정되지 않은 서버를 함께 알려주세요.

## 3단계: 사용 가능한 스킬 조사

이 플러그인에서 사용할 수 있는 분석 스킬을 나열합니다.

| 스킬 | 하는 일 |
|-------|-------------|
| **단일세포 RNA QC** | MAD 기반 필터링으로 scRNA-seq 데이터 품질을 관리 |
| **scvi-tools** | 단일세포 오믹스(scVI, scANVI, totalVI, PeakVI 등)를 위한 딥러닝 도구 모음 |
| **Nextflow 파이프라인** | nf-core 파이프라인 실행(RNA-seq, WGS/WES, ATAC-seq) |
| **Instrument Data to Allotrope** | 실험실 장비 출력을 Allotrope ASM 형식으로 변환 |
| **Scientific Problem Selection** | 연구 문제를 체계적으로 고르기 위한 프레임워크 |

## 4단계: 바이너리 MCP 서버

별도로 설치해 사용할 수 있는 두 개의 추가 MCP 서버도 함께 알려주세요.

- **~~게놈 플랫폼** - 클라우드 분석 데이터와 워크플로에 접근
  설치: https://github.com/10XGenomics/txg-mcp/releases 에서 `txg-node.mcpb`를 다운로드하세요.
- **~~도구 데이터베이스**(Harvard MIMS) - 과학적 발견을 위한 AI 도구
  설치: https://github.com/mims-harvard/ToolUniverse/releases 에서 `tooluniverse.mcpb`를 다운로드하세요.

이 바이너리 파일들은 다운로드가 필요하며 선택 사항입니다.

## 5단계: 도움 요청 방법

연구원에게 오늘 무엇을 하고 있는지 물어보세요. 흔한 워크플로를 바탕으로 시작할 만한 제안을 함께 제공합니다.

1. **문헌 검토** - "[주제]에 대한 최근 논문을 ~~문헌 데이터베이스에서 검색해 주세요"
2. **시퀀싱 데이터 분석** - "단일 세포 데이터에 대해 QC를 실행해 주세요" 또는 "RNA-seq 파이프라인을 설정해 주세요"
3. **신약 개발** - "~~[단백질]을 표적으로 하는 화합물을 ~~화학 데이터베이스에서 찾아 주세요" 또는 "[질병]에 대한 약물 표적을 찾아 주세요"
4. **데이터 표준화** - "내 기기 데이터를 Allotrope 형식으로 변환해 주세요"
5. **연구 전략** - "새 프로젝트 아이디어를 평가하는 데 도움을 주세요"

사용자의 답변을 기다린 다음, 적절한 도구와 스킬로 안내합니다.
