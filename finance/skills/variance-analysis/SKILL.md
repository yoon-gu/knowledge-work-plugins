# 차이 분석

**중요**: 이 스킬은 차이 분석 워크플로우를 지원하지만 재무 자문을 제공하지 않습니다. 모든 분석은 보고에 사용하기 전에 자격을 갖춘 재무 전문가의 검토를 받아야 합니다.

차이 분해 기법, 중요성 기준, 설명문 작성, 워터폴 차트 방법론, 예산 대 실적 대 전망 비교에 대한 내용입니다.

## 차이 분해 기법

### 가격 / 물량 분해

가장 기본적인 차이 분해입니다. 수익, 매출원가, 가격 x 물량으로 표현할 수 있는 모든 지표에 사용됩니다.

**공식:**
```
총 차이 = 실적 - 예산 (또는 전기)

물량 효과 = (실적 물량 - 예산 물량) x 예산 가격
가격 효과 = (실적 가격 - 예산 가격) x 실적 물량
믹스 효과 = 잔여(교호작용), 또는 비례 배분

검증:  물량 효과 + 가격 효과 = 총 차이
       (믹스가 가격/물량 항에 포함된 경우)
```

**3원 분해(믹스 분리):**
```
물량 효과 = (실적 물량 - 예산 물량) x 예산 가격 x 예산 믹스
가격 효과 = (실적 가격 - 예산 가격) x 예산 물량 x 실적 믹스
믹스 효과 = 예산 가격 x 예산 물량 x (실적 믹스 - 예산 믹스)
```

**예시 — 수익 차이:**
- 예산: 10,000개 x $50 = $500,000
- 실적: 11,000개 x $48 = $528,000
- 총 차이: +$28,000 유리
  - 물량 효과: +1,000개 x $50 = +$50,000 (유리 — 더 많이 판매)
  - 가격 효과: -$2 x 11,000개 = -$22,000 (불리 — 낮은 평균 판매가)
  - 순: +$28,000

### 요율 / 믹스 분해

단위 경제성이 다른 세그먼트 간 혼합 요율 분석 시 사용됩니다.

**공식:**
```
요율 효과 = Sum of (실적 물량_i x (실적 요율_i - 예산 요율_i))
믹스 효과 = Sum of (예산 요율_i x (실적 물량_i - 예산 믹스 기준 예상 물량_i))
```

**예시 — 매출총이익률 차이:**
- 제품 A: 60% 마진, 제품 B: 40% 마진
- 예산 믹스: A 50%, B 50% → 혼합 마진 50%
- 실적 믹스: A 40%, B 60% → 혼합 마진 48%
- 믹스 효과가 마진 2pp 축소를 설명

### 인원 / 보상 분해

급여 및 인건비 차이 분석 시 사용됩니다.

```
총 보상 차이 = 실적 보상 - 예산 보상

분해 항목:
1. 인원 차이       = (실적 인원 - 예산 인원) x 예산 평균 보상
2. 요율 차이       = (실적 평균 보상 - 예산 평균 보상) x 예산 인원
3. 믹스 차이       = 직급/부서 믹스 변화로 인한 차이
4. 시점 차이       = 계획 대비 조기/지연 채용(부분 기간 효과)
5. 이직 영향       = 계획에 없던 퇴직으로 인한 절감(충원 비용으로 부분 상쇄)
```

### 지출 카테고리 분해

가격/물량이 적용되지 않는 영업비용 분석 시 사용됩니다.

```
총 영업비용 차이 = 실적 영업비용 - 예산 영업비용

분해 항목:
1. 인원 연동 비용       (급여, 복리후생, 급여세, 채용비)
2. 물량 연동 비용       (호스팅, 거래 수수료, 수수료, 배송비)
3. 재량적 지출          (출장, 이벤트, 전문 서비스, 마케팅 프로그램)
4. 계약/고정 비용       (임차료, 보험, 소프트웨어 라이선스, 구독)
5. 일회성/비경상        (퇴직금, 법적 합의, 제각, 프로젝트 비용)
6. 시점/배분            (계획 대비 기간 간 이동한 지출)
```

## 중요성 기준 및 조사 트리거

### 기준 설정

중요성 기준은 조사 및 설명이 필요한 차이를 결정합니다. 다음을 기반으로 기준을 설정합니다:

1. **재무제표 중요성:** 일반적으로 주요 벤치마크(수익, 총자산, 당기순이익)의 1-5%
2. **항목 규모:** 큰 항목은 더 낮은 비율 기준 적용
3. **변동성:** 변동성이 큰 항목은 노이즈를 피하기 위해 더 높은 기준 필요할 수 있음
4. **경영진 관심:** 어느 수준의 차이가 의사결정을 바꾸겠는가?

### 권장 기준 프레임워크

| 비교 유형 | 금액 기준 | 비율 기준 | 트리거 |
|----------------|-----------------|---------------------|---------|
| 실적 vs 예산 | 조직별 상이 | 10% | 어느 하나 초과 |
| 실적 vs 전기 | 조직별 상이 | 15% | 어느 하나 초과 |
| 실적 vs 전망 | 조직별 상이 | 5% | 어느 하나 초과 |
| 전월 대비(MoM) | 조직별 상이 | 20% | 어느 하나 초과 |

*손익계산서 항목의 경우 수익의 0.5%-1%가 일반적 금액 기준입니다.*

### 조사 우선순위

여러 차이가 기준을 초과하는 경우 다음 순서로 조사 우선순위를 결정합니다:

1. **최대 절대 금액 차이** — P&L 영향이 가장 큰 항목
2. **최대 비율 차이** — 프로세스 문제 또는 오류를 나타낼 수 있음
3. **예상치 못한 방향** — 추세 또는 기대와 반대 방향의 차이
4. **신규 차이** — 정상이었다가 이탈한 항목
5. **누적/추세 차이** — 매 기간 증가하는 차이

## 차이 설명문 작성

### 각 차이 설명의 구조

```
[항목]: [유리/불리] 차이 $[금액] ([비율]%)
vs [비교 기준], [기간]

요인: [주요 요인 설명]
[2-3문장으로 차이의 사업적 사유를 설명하고, 기여 요인의
구체적 수치를 포함]

전망: [일회성 / 지속 예상 / 개선 중 / 악화 중]
조치: [불필요 / 모니터링 / 추가 조사 / 전망 업데이트]
```

### 설명문 품질 체크리스트

좋은 차이 설명문은 다음과 같아야 합니다:

- [ ] **구체적:** "예상보다 높음"이 아닌 실제 요인을 명시
- [ ] **수치화:** 각 요인의 금액 및 비율 영향 포함
- [ ] **인과적:** 무엇이 일어났는지뿐만 아니라 왜 일어났는지 설명
- [ ] **전향적:** 차이가 지속될 것으로 예상되는지 명시
- [ ] **실행 가능:** 필요한 후속 조치 또는 의사결정 식별
- [ ] **간결:** 2-4문장, 불필요한 내용 없음

### 피해야 할 일반적인 설명문 안티패턴

- "수익이 높은 수익 때문에 예산보다 높았습니다" (순환적 — 실제 설명 없음)
- "이번 기간에 비용이 증가했습니다" (모호 — 어떤 비용? 왜?)
- 무엇이 조기/지연되었고 언제 정상화될지 명시하지 않은 "시점"
- 항목이 무엇인지 설명하지 않은 "일회성"
- 중요 차이에 대해 "여러 소액 항목" (추가 분해 필요)
- 최대 요인에만 집중하고 상쇄 항목 무시

## 워터폴 차트 방법론

### 개념

워터폴(또는 브릿지) 차트는 일련의 양/음의 기여 요인을 통해 한 값에서 다른 값으로 이동하는 과정을 보여줍니다. 차이 분해를 시각화하는 데 사용됩니다.

### 데이터 구조

```
시작값:     [기준/예산/전기 금액]
요인:       [부호가 있는 기여 요인 목록]
종료값:     [실적/당기 금액]

검증:       시작값 + 모든 요인의 합 = 종료값
```

### 텍스트 기반 워터폴 형식

차트 도구를 사용할 수 없는 경우 텍스트 워터폴로 표시합니다:

```
워터폴: 수익 — Q4 실적 vs Q4 예산

Q4 예산 수익                                       $10,000K
  |
  |--[+] 물량 성장(신규 고객)                       +$800K
  |--[+] 확장 수익(기존 고객)                       +$400K
  |--[-] 가격 인하 / 할인                           -$200K
  |--[-] 이탈 / 축소                                -$350K
  |--[+] 환율 우호 효과                             +$50K
  |--[-] 시점(Q1으로 이연된 딜)                     -$150K
  |
Q4 실적 수익                                       $10,550K

순 차이: +$550K (+5.5% 유리)
```

### 브릿지 조정 테이블

워터폴을 보완하는 조정 테이블:

| 요인 | 금액 | 차이 대비 % | 누적 |
|--------|--------|---------------|------------|
| 물량 성장 | +$800K | 145% | +$800K |
| 확장 수익 | +$400K | 73% | +$1,200K |
| 가격 인하 | -$200K | -36% | +$1,000K |
| 이탈 / 축소 | -$350K | -64% | +$650K |
| 환율 우호 효과 | +$50K | 9% | +$700K |
| 시점(딜 이연) | -$150K | -27% | +$550K |
| **총 차이** | **+$550K** | **100%** | |

*참고: 상쇄 항목이 있을 경우 개별 요인의 비율이 100%를 초과할 수 있습니다.*

### 워터폴 모범 사례

1. 요인을 최대 양수에서 최대 음수 순서(또는 논리적 사업 순서)로 정렬
2. 최대 5-8개 요인 유지 — 소액 항목은 "기타"로 집계
3. 워터폴이 조정되는지 확인(시작 + 요인 = 종료)
4. 색상 코딩: 유리는 녹색, 불리는 빨간색(시각적 차트에서)
5. 각 막대에 금액과 간략한 설명 라벨 표시
6. "총 차이" 요약 막대 포함

## 예산 대 실적 대 전망 비교

### 3원 비교 프레임워크

| 지표 | 예산 | 전망 | 실적 | 예산 차이($) | 예산 차이(%) | 전망 차이($) | 전망 차이(%) |
|--------|--------|----------|--------|-------------|-------------|---------------|---------------|
| 수익 | $X | $X | $X | $X | X% | $X | X% |
| 매출원가 | $X | $X | $X | $X | X% | $X | X% |
| 매출총이익 | $X | $X | $X | $X | X% | $X | X% |

### 각 비교의 사용 시점

- **실적 vs 예산:** 연간 성과 측정, 보상 결정, 이사회 보고. 예산은 연초에 설정되며 일반적으로 변경되지 않음.
- **실적 vs 전망:** 운영 관리, 신규 이슈 식별. 전망은 현재 기대를 반영하여 주기적(월간 또는 분기)으로 업데이트.
- **전망 vs 예산:** 계획 수립 이후 기대치 변화 이해. 계획 정확도 문제 식별에 유용.
- **실적 vs 전기:** 추세 분석, 순차적 성과. 예산이 의미 없는 경우(신사업, 인수 후) 유용.
- **실적 vs 전년:** 전년 대비 성장 분석, 계절성 조정 비교.

### 전망 정확도 분석

계획 개선을 위해 시간 경과에 따른 전망 정확도를 추적합니다:

```
전망 정확도 = 1 - |실적 - 전망| / |실적|

MAPE (평균 절대 백분율 오차) = 기간별 |실적 - 전망| / |실적|의 평균
```

| 기간 | 전망 | 실적 | 차이 | 정확도 |
|--------|----------|--------|----------|----------|
| 1월    | $X       | $X     | $X (X%)  | XX%      |
| 2월    | $X       | $X     | $X (X%)  | XX%      |
| ...    | ...      | ...    | ...      | ...      |
| **평균**|          |        | **MAPE** | **XX%**  |

### 차이 추세

연중 차이 변화를 추적하여 체계적 편향을 식별합니다:

- **지속적 유리:** 예산이 너무 보수적일 수 있음(샌드백)
- **지속적 불리:** 예산이 너무 공격적이거나 실행 문제
- **증가하는 불리:** 성과 악화 또는 비현실적 목표
- **축소되는 차이:** 연중 전망 정확도 개선(정상 패턴)
- **변동적:** 예측 불가한 사업 또는 열악한 전망 방법론
